# 歷屆篩選測驗出卷系統

讓老師透過公開網址選擇年度、年級、科目與題型，挑選題目後在瀏覽器下載 Word 試卷。

## 目前狀態

- 國語科 108～114 年、一～六年級已匯入，共 896 個可選項目。
- 公開端為純靜態網站，Word 直接在老師的瀏覽器產生。
- D 槽原專案保留不動；本資料夾是 Codex 後續維護的主要工作區。
- 四～六年級已建立批次裁切、分類與圖片引用流程；目前仍在依使用者實際畫面逐批驗收。

## 目錄

- `private/legacy-app/`：私人維護端，放置原始 PDF、裁切程式與完整工作資料；不公開部署。
- `public/`：部署至 Firebase Hosting 的公開網站；不得放入管理功能、原始 PDF或敏感資料。
- `docs/`：架構決策、資料規格與發布流程。

## 四～六年級分類規則

| 年級 | 字詞_認識 | 字詞_應用 | 句段_閱讀 | 篇章_閱讀 |
| --- | --- | --- | --- | --- |
| 四年級 | 1～7 | 8～12 | 13～19 | 20～25（2 組） |
| 五年級 | 1～5 | 6～10 | 11～17 | 18～25（3 組） |
| 六年級 | 1～5 | 6～10 | 11～17 | 18～25（3 組） |

規則集中在 `private/legacy-app/auto_crop.py` 的 `GRADE_QUESTION_RULES`，裁切、分類與閱讀題組打包共用同一份設定。

## 題圖縮邊與 Word 排版

- 題圖只裁右側空白；左側、上側、下側與高度保持不變。
- 右側保留約 18 像素安全邊距；密集表格線、圖表與插圖一律視為內容，不再當成欄線忽略。
- 題幹、選項、表格與插圖必須完整保留，替換前使用像素比對與視覺對照驗證。
- Word 使用 B4 直式雙欄；每張圖左對齊，依原始約 1000 像素欄寬維持固定縮放比例，僅在超過單欄時縮小，避免字體忽大忽小。
## 新增年度流程

1. 將新年度 PDF 放入私人端 `pdfs/`。
2. 在暫存區自動裁切，不直接覆蓋正式題庫。
3. 檢查 25 題是否齊全、分類分布、圖片引用、空白與過短裁切。
4. 產生縮圖總覽，人工核對文章、圖表、題幹與選項邊界。
5. 只把已引用且通過檢查的圖片與題目資料合併到公開端。
6. 本機預覽 Word 下載，驗收後才發布 Firebase Hosting。

## 安全規則

- 不儲存學生姓名、電子信箱或其他可識別資料。
- 不公開原始 PDF、私人裁切後台或本機路徑。
- 不提交 `.env`、token、密碼、服務帳戶金鑰或 Admin SDK 憑證。
- 公開端不提供上傳、刪除或修改題庫的 API。
- 發布前必須驗證題庫數量、圖片引用及 Word 匯出。

## 部署

預定使用 Firebase Hosting。正式網址與部署指令會在公開版完成驗收後補上。
